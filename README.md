# Unofficial port of Slimefun mod

## Description

This is an unofficial NeoForge port of the Slimefun mod, adding a tech and magic item system to Minecraft 1.21.1+. Slimefun is an open-source mod that allows players to build and use multiblock machines to craft advanced items.

This mod is currently in early development, and the author is a beginner, please be patient and understanding, thank you.

## Features

- Tech items: solar panels, electric motors, etc.
- Magic items: magic books, enchanted tools, etc.
- Automation systems: machines and pipes
- Compatible with NeoForge loader

## Installation

1. Ensure you have Minecraft 1.21.1 or higher.
2. Download and install [NeoForge](https://neoforged.net/).
3. Download the mod JAR file.
4. Place the JAR file in the `.minecraft/mods` folder.
5. Launch the game.

### Dependencies

- NeoForge 47.1.0+
- Minecraft 1.21.1+

## Usage

After launching the game, you can use Slimefun items in creative or survival mode. Check the in-game guide or Wiki for more information.

## Building

### Requirements

- Java 21+
- Gradle 8.0+

### Steps

1. Clone the repository:
   ```
   git clone https://github.com/AlerCello86767/neoforgemod_slimefun.git
   cd neoforgemod_slimefun
   ```

2. Build the mod:
   ```
   ./gradlew build
   ```

3. Output files are located in `build/libs/`.
